2. Sphinx Doc Environment

2.1. Install Python

Install Python following the instructions in: Python Environment.

Quick guide:

  • Go to the official Python download page and choose the version suitable for your system.

  • Run the installer by double-clicking it and follow the on-screen prompts. Be sure to check the “Add Python to PATH” option, which will automatically update your PATH environment variable.

  • Click “Install Now” for a standard installation, or choose “Customize installation” if you’d like to modify settings such as the installation location and advanced options.

  • To verify Python installation, run the following commands:

    1# verify Python installation
2python --version
3python -V

2.2. Create Sphinx-Doc Project

  1. Create the Sphinx-Doc folder <doc>.

  2. Navigate to the <doc> folder.

  3. Create and Activate Python virtual environment

    1python -m venv .venv
2# Activate the Python virtual environment - Windows
3.venv\Scripts\activate.bat

  4. Install Sphinx-Doc packages.

    1# Installing py packages
2pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -U sphinx sphinx_rtd_theme myst-parser sphinx-autobuild sphinx-design

2.3. Sphinx-Doc Initialization

  1. Run sphinx-quickstart <doc-folder>

  2. Generate html build

  3. Run command:

     1# interactive
 2sphinx-quickstart docs
 3# Non-interactive
 4sphinx-quickstart --sep --project="<project-name>" --author="<author-name>" -v "<version>" --quiet docs
 5# run html build:
 6#    1. sphinx-build -M html docs/source/ docs/build/
 7#       # OR
 8#    2. cd docs && make clean && make html
 9cd docs
10make clean && make html

2.5. Document Structures

  1. ########### with overline, for parts

  2. *********** with overline, for chapters

  3. =========== for sections

  4. ----------- for subsections

  5. ^^^^^^^^^^^ for subsubsections

  6. """"""""""" for paragraphs
